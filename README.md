# DropTrack – Web3 Airdrop Tracker

DropTrack is a simple Web3 discovery tool that tracks new blockchain projects, airdrops, and community tasks.

## Goal

Many users miss early Web3 opportunities.  
DropTrack helps organize project updates, tasks, and airdrop programs in one place.

## Features

- Track new Web3 projects
- View airdrop tasks
- Follow testnet programs
- Discover early opportunities

## Tech Stack

- HTML dashboard
- Node.js script
- JSON dataset

## Future Work

- Add API for project discovery
- Integrate decentralized storage
- Improve filtering and alerts
